# LabZZZ-Blog

## A simple blog as a homework after a dure auto-apprentisage

## Basic Features
* Users login, register
* Categorize articles
* Content moderation
* Basic read and comment fonction 

## Server Requirements
* PHP 8.1.5 + 10.4.24-MariaDB + Apache/2.4.53
* PDO PHP Extension
